# :books: BibTeX-Cleaner

BibTeX-Cleaner is a Python command-line tool that analyzes a `.bib` file against the citations actually used in a LaTeX document (via its `.aux` file). It reports unused and missing bibliography entries, and can generate a cleaned `.bib` file containing only the citations that are actually referenced in your paper.

## :mag: Description <a name="description"/>
When writing academic papers, `.bib` files often accumulate entries that are never cited, or references get renamed/removed causing broken citations. This tool parses the `\citation{}` commands from a LaTeX `.aux` file (including BibLaTeX's `\abx@aux@cite` and `\abx@aux@segm` commands) and cross-references them against the entries in a `.bib` file.

Key features:
* Detects **unused entries** (present in `.bib` but never cited).
* Detects **missing entries** (cited in the paper but not found in `.bib`).
* Generates a **clean `.bib` file** containing only the used citations.
* Optionally overwrites the original `.bib` file, keeping only used entries.

## :gear: Setup <a name="setup"/>
Requires Python 3.7+ (no external dependencies, uses only the standard library).

Clone or download the script, then make it executable (optional):

```
chmod +x bib_cleaner.py
```

## :man_technologist: How to use <a name="how-to-use"/>
Run the script passing the `.aux` and `.bib` files as command-line arguments:

```
python3 bib_cleaner.py output.aux bibliography.bib
```

By default, the cleaned bibliography is saved as `references_clean.bib`. You can specify a custom output path with `-o` / `--output`:

```
python3 bib_cleaner.py output.aux bibliography.bib -o my_clean_refs.bib
```

**Arguments:**
| Argument | Description |
|---|---|
| `aux_file` | Path to the `.aux` file (generated by LaTeX after compilation) |
| `bib_file` | Path to the `.bib` file to analyze |
| `-o`, `--output` | Path for the cleaned `.bib` output (default: `references_clean.bib`) |
| `--generate-clean-bib`/`--no-generate-clean-bib` | Generate a cleaned .bib file with only used entries (default: True)|
| `--remove-unused`/`--no-remove-unused` |  Overwrite the original .bib file removing unused entries (default: False)|

> :bulb: The `.aux` file must be up to date. Run `pdflatex` on your `.tex` document at least once before using this tool, so all `\cite{}` commands are reflected in the `.aux` file.

In Overleaf, you can access the generated .aux file after compilation by opening the "Logs and output files" menu, then selecting "Other logs and files". From there, locate and open the corresponding .aux file.

If the .aux file is missing or outdated, the tool may not be able to detect or process the citations correctly. Recompile the document and try again.
